# Maven-Demos
This is a simple demo for working with Maven  <br>
This is a lab repo for the Coursera course [Master and Apply Maven for Java Build Automation](https://www.coursera.org/learn/master-apply-maven-java-build-automation) <br>

This lab repo focuses on the following main sections: <br>

### Maven in five minutes 

Create a simple project on the command line: <br>
`mvn archetype:generate "-DgroupId=com.mycompany.app" "-DartifactId=my-app" "-DarchetypeArtifactId=maven-archetype-quickstart" "-DarchetypeVersion=1.5" "-DjavaCompilerVersion=25" "-DinteractiveMode=false"`
